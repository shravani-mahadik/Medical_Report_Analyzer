# 🏥 Medical Report Analyzer

An AI-powered Medical Report Analyzer built using **Python**, **Streamlit**, **EasyOCR**, and **ReportLab**.

## ✨ Features

- 📄 Upload PDF/Image medical reports
- 🔍 OCR-based text extraction
- 📊 Detect medical parameters
- ❤️ Calculate Health Score
- 📈 Analytics Dashboard
- 📂 Report History
- 📉 Health Trend Analysis
- 📥 Download analyzed report as PDF

---

## 🛠 Tech Stack

- Python
- Streamlit
- EasyOCR
- ReportLab
- Pandas
- OpenCV
