# 🩺 Medical Report Analyzer

An AI-powered web application built using Python and Streamlit that analyzes medical reports, extracts health parameters, detects abnormal values, and generates an easy-to-understand summary.

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- PDFPlumber
- EasyOCR
- Plotly

## Features
- Upload PDF/Image Reports
- Extract Medical Parameters
- Detect Abnormal Values
- AI Summary
- Interactive Dashboard
